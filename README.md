# 차문지기 (CarMoonJiGi)

차량 접근자 얼굴인식⦁모션 인식⦁BLE 근접 인증 기반 자동차 출입 보안 강화 플랫폼 개발

## 프로젝트 소개

BLE 스마트키, 얼굴인식, 손 제스처 인식, 지문인식을 활용한 다중 생체인증 기반 차량 출입 및 차량 내 결제 시스템

## 주요 기능

- BLE 기반 사용자 접근 감지
- 얼굴 인식
- 손 제스처 인증
- 지문 인증
- 차량 도어 제어
- 차량 내 결제 시뮬레이션

## 개발 환경

- Python
- Raspberry Pi 5
- OpenCV
- Face Recognition
- MediaPipe
