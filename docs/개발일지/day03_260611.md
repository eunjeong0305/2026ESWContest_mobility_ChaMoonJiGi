## 얼굴 검출 테스트

- 정면 얼굴 인식 성공
- 측면 얼굴 일부 인식 성공
- 얼굴 기울기 증가 시 인식률 감소 확인
- Haar Cascade 모델의 한계 확인

## 얼굴 검출 성능 개선

- minSize=(80,80) : 의류 영역 오검출 발생
- minSize=(100,100) : 얼굴 검출과 오검출 방지 균형 확인
- minSize=(150,150) : 작은 얼굴 검출 실패

결론 : minSize=(100,100) 적용

## 얼굴 인증 프로토타입 구현

- USB 웹캠 연결
- 웹캠 실시간 영상 획득

- 얼굴 검출 시 AUTH SUCCESS/DOOR OPEN 표시
- 얼굴 미검출 시 AUTH FAIL/LOCKED 표시
- 얼굴 영역 실시간 표시
