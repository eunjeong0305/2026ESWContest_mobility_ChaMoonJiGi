# 차문지기 개발일지 - 1주차

## 개발 기간
2026.06.08 ~ 2026.06.15

## 진행 내용
- GitHub Repository 생성
- 프로젝트 폴더 구조 설계
- VS Code 설치
- Python 3.13 설치
- OpenCV 설치
- OpenCV를 이용하여 이미지 출력에 성공하였다.
- Haar Cascade 방식으로 얼굴 검출 시도
- DNN(res10_300x300_ssd) 모델로 얼굴 검출 성공

## 문제점
- Python 명령어가 인식되지 않음
- OpenCV 카메라 테스트 실패
- Haar Cascade가 얼굴 대신 귀를 인식하는 오류 발생
- DNN 모델 파일 경로 오류 발생

## 해결 방법
- py 명령어 사용
- 웹캠이 없는 환경임을 확인
- Haar Cascade 한계로 판단, DNN 방식으로 교체
- 모델 파일(deploy.prototxt, caffemodel)을 프로젝트 루트에 다운로드 후 경로 수정
